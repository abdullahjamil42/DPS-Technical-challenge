# DPS-Technical-challenge
This repository contains my submission for the Digital Product School (DPS) Software Engineer Challenge. The objective was to build a robust, user-centric application that delivers a seamless, interactive user experience within a strict one-week timeline. Built using a modern full-stack architecture, the solution prioritizes clean code separation, scalable components, and reliable data flow. To mirror a production-ready engineering workflow, the entire development process was broken down into incremental feature milestones, documented through a transparent, structured, and organic Git commit history.
